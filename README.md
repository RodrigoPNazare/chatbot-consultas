Chatbot de Consultas

Este é um projeto desenvolvido em Node.js que utiliza APIs externas para realizar consultas de cotação de moedas e previsão do tempo em tempo real.

Funcionalidades
- **Consulta de Moeda:** Busca a cotação atual de diversas moedas em relação ao Real (BRL).
- **Consulta de Clima:** Informa a temperatura atual de qualquer cidade desejada.

 Tecnologias Utilizadas
- Frontend: HTML5, CSS3 (Animate.css) e JavaScript (Fetch API).
- Backend: Node.js com Express.
- Integrações: AwesomeAPI e wttr.in.
- Outros: Axios, CORS.

 Como Executar

### Pré-requisitos
Certifique-se de ter o Node.js instalado na sua máquina.

 Passos
1. Clone este repositório:
git clone https://github.com/RodrigoPNazare/chatbot-consultas.git

2. Entre na pasta do projeto:
cd chatbot-consultas

3. Instale as dependências:
npm install

4. Inicie o servidor:
node index.js

5. Com o servidor rodando em http://localhost:3000, abra o arquivo index.html em seu navegador.
